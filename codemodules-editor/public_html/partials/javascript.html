Muuttujat ja laskenta

Muuttujaan voidaan säilöä erilaisia arvoja, kuten x = 5;, y = 4; tai z = true;. Myöhemmin näitä samoja arvoja voidaan muuttaa myöhemmin, esimerkiksi x = 4; tai y = x + 1. Jälkikkäisessä esimerkissä muuttujan y arvo riippuu muuttujan x arvosta suoritushetkellä. Tässä tapauksessa siis y saisi arvon 5. Huomaa, että yn arvo ei muutu, vaikka xn arvo muuttuisi.

Lukuja voi laskea normaalisti: 1+5, 7-2, 5/3 sekä 3*2 toimivat kuten luulisitkin. Usein laskennan tulos tallennetaan muuttujaan, eli luku = 1+5; tai muuttujaa käytetään laskennan osana: luku = luku / 2;.

Muuttujalla on nimi, esimerkiksi yllä esitetyjen muuttujien nimet ovat x, y ja luku. Muista nimetä muuttuja selkeästi sen tarkoituksen mukaisesti.

Funktiot ja paluuarvot

Ohjelmakielissä on usein tarpeen toistaa samaa rutiinia pienillä muutoksilla. Tätä varten ohjelmakoodissa on mahdollisuus tehdä funktiota:

function kerroKahdella(sisään){
  var ulos = sisään * 2;
  return ulos; }

sisään on muuttuja, jota funktio käsittelee - näitä voi olla myös useita pilkuilla eroteltuna. Tehtyään tarvittavat laskutoimitukset, funktio palauttaa luvun ulos (tai, minkä tahansa asian joka on ennen return-sanaa).

Ehtolause

Joskus on tarpeen suorittaa tiettyjä komentoja vain ehdon ollessa voimassa. Tähän käyttöön on olemassa if-lause:

if( x == 5 ) {
   x = 0;
}

Nyt muuttuja x saa arvokseen nollan jos ja vain jos x on viisi. Ehtoina voi käyttää matematiikasta tuttuja merkkejä: <, > sekä == (huomaa: kaksi yhtäsuuruusmerkkiä).
